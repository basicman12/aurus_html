    Решения для Отелей Решения для Сall-центров
    • RichCall

      Видеозвонок с сайта. Совместно работайте над документами и обменивайтесь ими с клиентами сайта.

    • MeetMe

      Планирование ВКС-совещаний из MS Outlook на ВКС-серверах Cisco Telepresence MCU.

      Послать документ по факсу теперь также просто, как отправить его по электронной почте.

      Автоматический обзвон клиентов по телефону.
